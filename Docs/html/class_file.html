bool File::ProcessFile ( )

Process the input file.

Opens the input file, if file was not opened correnctly, return false, otherwise reads first line, which contains count of the rest of the lines. Then we read the file line by line, where we split the file using whitespaces. All the values ve got from single line, we store as dimensions in vector, and when finished reading single line, push the dimensions vector into vector

Returns
True, if file was processed correctly, otherwise false.
